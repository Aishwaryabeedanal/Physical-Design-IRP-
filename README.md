# Physical Design of UART with RISC-V Integration (CDAC IRP Project)

## 🔹 Overview
This project focuses on the complete RTL-to-GDSII physical design of a UART module and its integration with a RISC-V processor for serial communication.

## 🔹 Key Features
- Full RTL-to-GDSII physical design flow
- UART design and integration with RISC-V
- Timing, power, and area optimization
- Design closure with analysis reports

## 🔹 Description
The project involves implementing the complete physical design flow for a UART module, including synthesis, floorplanning, placement, clock tree synthesis (CTS), and routing.

The UART IP is integrated with a RISC-V processor to enable serial communication. Post-layout analysis was performed to evaluate timing, power, and area, ensuring successful design closure.

## 🔹 Physical Design Flow
- Synthesis  
- Floorplanning  
- Placement  
- Clock Tree Synthesis (CTS)  
- Routing  

## 🔹 Tools Used
- Industry-standard EDA tools (CDAC IRP Flow)
- Verilog HDL
